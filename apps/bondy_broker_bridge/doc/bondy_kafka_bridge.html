    validate_action(Action0)

    View Source

    Validates the action specification. An action spec is a map containing the following keys:

    * type :: binary() - <<"produce_sync">>. Optional, the default value is <<"produce_sync">>. * topic :: binary() - the Kafka topic we should produce to. * key :: binary() - the kafka message's key * value :: any() - the kafka message's value * options :: map() - a map containing the following keys

         erlang
  #{
      <<"type">> <<"produce">>,
      <<"topic": <<"com.magenta.wamp_events",
      <<"key": "\"{{event.topic}}/{{event.publication_id}}\"",
      <<"value": "{{event}}",
      <<"options" : {
          <<"client_id": "default",
          <<"acknowledge": true,
          <<"required_acks": "all",
          <<"partition": null,
          <<"partitioner": {
              "algorithm": "fnv32a",
              "value": "\"{{event.topic}}/{{event.publication_id}}\""
          },
          <<"encoding">>: <<"json">>
      }
  }
