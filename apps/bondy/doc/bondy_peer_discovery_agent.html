    bondy_peer_discovery_agent behaviour (bondy v1.0.0-beta.16) View Source

    This state machine is reponsible for discovering Bondy cluster peers using the defined implementation backend (callback module).

    Its behaviour can be configured using the cluster.peer_discovery family of bondy.conf options.

    If the agent is enabled (cluster.peer_discovery.enabled) and the Bondy node has not yet joined a cluster, it will lookup for peers using the defined implementation backend callback module (cluster.peer_discovery.type).
