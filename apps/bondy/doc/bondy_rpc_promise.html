    bondy_rpc_promise (bondy v1.0.0-beta.16) View Source

    A promise is used to implement a capability and a feature: - the capability to match the callee response (wamp_yield() or wamp_error()) back to the origin wamp_call() and Caller - the call_timeout feature at the dealer level

    enqueue(RealmUri, Bondy_rpc_promise, Timeout)

    View Source

    Adds the invocation promise P to the promise queue for realm RealmUri using a timeout of Timeout.

    If the promise is not dequeued before Timeout milliseconds, the caller will receive an error with reason "wamp.error.timeout".
