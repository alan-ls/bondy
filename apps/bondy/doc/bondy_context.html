    bondy_context (bondy v1.0.0-beta.16) View Source

    A Bondy Context lets you access information that defines the state of an interaction. In a typical interacion, several actors or objects have a hand in what is going on e.g. bondy_session, wamp_realm, etc.

    The Bondy Context is passed as an argument through the whole request-response loop to provide access to that information.
