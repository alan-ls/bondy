    bondy_rbac_group (bondy v1.0.0-beta.16) View Source

    **Note:** Usernames and group names are stored in lower case. All functions in this module are case sensitice so when using the functions in this module make sure the inputs you provide are in lowercase to. If you need to convert your input to lowercase use string:casefold/1.

Otherwise returns Groups unmodified. Fails with {cycle, Path :: [name()]} exception if the graph directed graph has cycles of length two or more.
