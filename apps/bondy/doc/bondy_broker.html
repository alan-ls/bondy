    subscribe(RealmUri :: uri(), Opts :: map(), Topic :: uri(), SubscriberOrFun :: pid() | function()) ->
             {ok, id()} | {ok, id(), pid()} | {error, already_exists | any()}.

    For internal use. If the last argument is a function, spawns a supervised instance of a bondy_subscriber by calling bondy_subscribers_sup:start_subscriber/4. The new process, calls subscribe/4 passing its pid as last argument.

    If the last argument is a pid, it registers the pid as a subscriber (a.k.a a local subscription)
